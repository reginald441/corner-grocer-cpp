# Corner Grocer Application (C++)

This project is a grocery tracking application that analyzes item purchase frequency using file input/output and object-oriented programming.

## Features
- Reads grocery purchase data from file
- Tracks item frequency
- Displays histogram and counts
- Uses OOP design (classes and functions)

## Technologies
- C++
- File Handling
- Object-Oriented Programming (OOP)
